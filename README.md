## INSTALLATION
 
 #### LINUX
 
    python3 -m pip install -r linux_requirements.txt

   *Other dependance* : __[xclip](https://github.com/astrand/xclip)__
    
------------------------------------------------

 #### Windows
 
    python -m pip install -r windows_requirements.txt


## RUN 

    python main.py
